Default position is static
relative position - lays out all elements as though the element were not positioned, and then adjust the element's position, without changing layout (and thus leaving a gap for the element where it would have been had it not been positioned).
Example in blue, the gap where the span was is present and the span is relocated relative to the gap

absolute position - Do not leave space for the element. Instead, position it at a specified position relative to its closest positioned ancestor or to the containing block. Absolutely positioned boxes can have margins, they do not collapse with any other margins.
Example in red where div3 position is default static so the position is relative to the main body
Example in green where div4 position is relative without change in top or left, so the span is absolute relative to the div instead of the body

fixed postion - Do not leave space for the element. Instead, position it at a specified position relative to the screen's viewport and don't move it when scrolled. When printing, position it at that fixed position on every page. This value always create a new stacking context.
Example in pink is always at the top of the screen, no matter the scrolling
